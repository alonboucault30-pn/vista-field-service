Vista Field Service v2 - Technical Inspection Platform

Como rodar:
1) Abra o CMD dentro desta pasta.
2) Rode: pip install -r requirements.txt
3) Rode: py -m streamlit run app.py

Melhorias v2:
- Tela inicial com botões.
- Botão Início em todas as abas.
- Botão Calcular antes de salvar.
- Componentes simples, duplos, triplos e quadruplos.
- Tabela padrão 30, 80, 125, 180, 250 e 300 bar.
- Vazão preenchida por pressão e eficiência calculada.
- Campo de óleo.
- Histórico com filtro e exclusão.
- Dashboard com pressão x vazão e pressão x eficiência.
- PDF com gráficos, tabelas e fotos.
